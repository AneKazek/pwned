# **App Name**: Dzaky Hax

## Core Features:

- Defacement Display: Display the 'Hacked By Muhammad Dzaky Haidar' message prominently.
- Glitchy Background: Implement a dark and glitchy background effect as demonstrated here: https://karthikdude.github.io/Deface/
- Hacker Graphic: Show a hacker-themed graphic like a skull or matrix code effect.
- Secondary Message: Include a secondary message 'You've been owned!' on the page.

## Style Guidelines:

- Primary color: Deep gray (#212121) for a dark, foreboding atmosphere.
- Background color: Black (#0A0A0A) to maintain a strong dark presence.
- Accent color: Vibrant red (#FF3333) and electric green (#33FF33) for the text elements.
- Body and headline font: 'Space Grotesk' sans-serif for a modern, tech-oriented headline style and glitch aesthetics.
- Utilize vector graphics for hacker-themed visuals such as skulls or code matrix designs.
- Use subtle glitch animations on the text or hacker graphics to enhance the visual effect.
